ungit

配置 ungit

1. 進入 http://mde.tw/wcm2019/content/實習項目二.html  ，下載 kmol_2019.7z 和  nodejs_with_ungit.7z 。

2. 兩者分別解開壓縮檔案，然後將 nodejs 放入可攜 data 目錄中。

3. 設定命令列搜尋路徑

利用SciTE 編輯器編輯 start_cadlab.bat(及start.bat)。

路徑指令: set path8=%Disk%:\nodejs;%Disk%:\nodejs\appdata\roaming\npm;

並且將 path8 納入 path 設定

4. 設定 USERPROFILE 環境變數

5. 若要啟動 ungit，必須使用下列指令:

ungit --launchBrowser=0 --launchCommand "C:\PROGRA~2\Google\Chrome\Application\chrome.exe %%U"
